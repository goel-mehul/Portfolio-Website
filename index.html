Project

Multi-Agent Research Scientist

6-agent LangGraph pipeline for autonomous ML paper research, synthesis, and report generation. Deployed at agentic-ai-project-mehul-goel.vercel.app

Architecture
  • 6-agent LangGraph pipeline: Planner → Searcher → Fetcher → Extractor → Synthesizer → Writer
  • Extractor agent normalizes full paper abstracts into structured summaries before synthesis — key architectural choice to preserve signal-to-noise ratio
  • ChromaDB vector store for semantic retrieval; FastAPI backend; React frontend on Vercel
  • Zero hallucinations constraint across full report length

Project

Robust RAG Under Semantic Drift

Hybrid retrieval pipeline stress-tested against semantic drift on climate change corpora — studying degradation and mitigation strategies.

Key Contributions
  • Hybrid retrieval: FAISS (dense) + BM25 (sparse) fused via Reciprocal Rank Fusion (RRF)
  • Identified that abstracts-only retrieval is higher signal-to-noise than full-paper retrieval
  • Designed and ran controlled experiments simulating semantic shift across time-series document corpora
  • Quantified retrieval precision degradation and proposed mitigation: dedicated Extractor agents

Project

RetailRL — Multi-Agent Pricing

Full PettingZoo AEC environment simulating 10 competing grocery retailers with DQN, PPO, A2C, and Q-Table agents. v1.0.0 released.

Environment Details
  • 266-dimensional observations, 50 products, 7 price actions per product per agent
  • MNL (Multinomial Logit) demand model, inventory manager, supply chain, promotion calendar
  • 10 reward functions supporting diverse agent objectives
  • 124 passing tests across 25 commits; CLI training package for reproducibility
  • Streamlit dashboard with muted dark theme; deployed on Streamlit Cloud
